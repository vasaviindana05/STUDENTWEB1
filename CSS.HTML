WHAT IS CSS?

    • CSS(CASCADING STYLING SHEETS) is a language used to style and format the apperance of web pages and it works alongside HTML to define how elements on webpage and displayed across various devices and media and CSS allows developers to control the layout,color,fonts,etc and overall design of awebsite efficiency

PROS AND CONS

PROS

    • The major benefits of using CSS is separation of content and design
    • meaning,CSS separates html structure from styling , making the code cleaner and easier to maintain
    • CSS has consistency across pages meaning, a single css file can be applied to multiple pages,ensuring a uniform look and feel

CONS

    • Apart from pros , there some cons in cons i.e there are no built in logic like no conditions ,loops or functions
    • There is a thing that need to be consider that it is quite hard to maintain because sometimes styles may overlap
    • CSS doesn't have proper structure and while integrating it lead ous to confusion
